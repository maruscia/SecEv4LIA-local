# Installation

Get SecEv4LIA up and running in seconds.

## Quick Install

### Requirements

- **Python 3.10+**
- **pip** or **uv** package manager

### Production Installation

<Tabs>
  <TabItem value="uv" label="uv (Recommended)" default>

```bash
uv add secev4lia
```

  </TabItem>
  <TabItem value="pip" label="pip">

```bash
pip install secev4lia
```

  </TabItem>
  <TabItem value="pip-git" label="pip (from GitHub)">

```bash
pip install git+https://github.com/AISecurityLab/SecEv4LIA.git
```

  </TabItem>
</Tabs>

### Verify Installation

After installation, verify everything works:

```bash
secev --version
```

### Initial Setup

SecEv4LIA works **out of the box without any account or API key**. All results are stored locally in a SQLite database at `~/.local/share/secev4lia/secev4lia.db`.

Run the interactive setup wizard to configure preferences:

```bash
secev init
```

This will guide you through:
1. **Set verbosity level** — Control logging detail (0=ERROR to 3=DEBUG)
2. **Save configuration** — Stored in `~/.config/secev4lia/config.json`

:::info Fully local
SecEv4LIA runs entirely offline. Results are stored in `~/.local/share/secev4lia/secev4lia.db` and are fully accessible via `secev results list`, the TUI, and the local web dashboard (`secev web`). No data is sent to any remote server.
:::

## Development Installation

For development or to access the latest features:

<Tabs>
  <TabItem value="uv-dev" label="uv (Recommended)" default>

```bash
# Clone the repository
git clone https://github.com/AISecurityLab/secev4lia.git
cd secev4lia

# Install with uv
uv sync --group dev
```

  </TabItem>
  <TabItem value="pip-dev" label="pip">

```bash
# Clone the repository
git clone https://github.com/AISecurityLab/secev4lia.git
cd secev4lia

# Install in development mode
pip install -e ".[dev]"
```

  </TabItem>
</Tabs>
